# Android-Hiring-Automation
Automate android hiring via ui tests and code quality tools.

This is a fun project with basic automation for android hiring.

Steps:
1. Create a base project with all the things setup such as UI test (idling resource) and findbugs.
2. Fork the repo - fill the code in the relevant place.
3. Send a PR. It will trigger travis ci. Get the signals about code quality and correctness based on the number of edge cases passed.
4. Profit :)
